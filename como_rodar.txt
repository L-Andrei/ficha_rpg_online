Para rodar o código basta ter instalado o python e instalar estas biblitecas:

pip install flask_session
pip install flask
pip install datetime
pip install flask_cors
pip install flask_sqlalchemy

Apos instaladas basta rodar o arquivo backend.py ir até o navegador e colocar o link:

http://locahost:5000/

A pasta HTML_css contém uma cópia das páginas, porem não necessita rodar o beckend para elas funcionarem, mas terão funções limitadas.
